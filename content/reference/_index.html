To build this workshop, I borrowed ideas and codes from several sources. The ones I can remember can be found below:

  • Data Visualization: A practical introduction by Kieran Healy

  • Fundamentals of Data Visualization by Claus Wilke

Courses

  • Introduction to Data Science by Eric Dunford
